## Project Overview

A static website for washing instructions, serving PDFs in English (`docs/en.pdf`) and Norwegian (`docs/no.pdf`). The `index.html` is the entry point.

## Architecture

- **`index.html`** — main landing page
- **`docs/en.pdf`** — English washing instructions
- **`docs/no.pdf`** — Norwegian washing instructions

No build system, package manager, or backend — pure static files intended to be served directly.
